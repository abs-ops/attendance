#3、方案设计
##3.1、设计原则
###3.1.1、先进性原则

&emsp;&emsp;采用先进的设计思想，选用先进的网络设备，采用科学的安装工艺，使网络在今后一定时期内保持技术上的先进性。

&emsp;&emsp;组网技术采用目前最为先进的基于IEEE802.11a/g/b标准的无线联网技术。
###3.1.2、开放性原则

&emsp;&emsp;网络设计及网络设备选型遵从国际标准及工业标准，使网络具有开放性和兼容性。
&emsp;&emsp;我们采用的无线产品是经过国际Wi－Fi组织认证的标准产品，可以和其它标准产品很好地兼容。

###3.1.3、可伸展性原则

&emsp;&emsp;网络设计在充分考虑当前情况的同时，必须考虑到今后较长时期内业务发展的需要，留有充分的升级和扩充的可能性。

###3.1.4、安全性原则
&emsp;&emsp;为了防止来自网络内部和外部的各种非法攻击和访问，设计中必须考虑安全措施。

###3.1.5、可靠性原则

&emsp;&emsp;网络系统的设计必须贯彻可靠性原则，使网络系统具有很高的可用性。

###3.1.6、可管理性原则

&emsp;&emsp;网络系统应具有良好的可管理性，使得网络管理人员能方便及时地掌握诸如网络拓扑结构、网络性能统计、网络故障等信息，能简便地对网络进行配置和调整，确保网络工作在良好状态。

##3.5. 设备介绍
1、ZG-5000-I IEEE 802.11g 54Mbps无线室外网桥

该网桥采用一体化户外设计（内置15dBi平板定向天线），减少了传统网桥连接天线RF馈缆的损耗，提高了无线网桥的整体性能。

##3.6. 安全策略
为了保证无线网络不受外来的非法访问和攻击，在安全方面我们设计了SSID、MAC过滤和128位WEP加密“三级”安全控制机制，使得内部网络具备较高的安全性。

##3.8. 方案特点
（1）良好的扩展性
由于采用无线技术，今后随着用户量的增加，无需线挖沟布线，只需增加相应的无线网桥、无线接入点或无线网卡即可，非常方便快捷。
（2）完善的安全措施
SSID、MAC过滤和128位WEP加密“三级”安全控制机制，保护内部网络不受外来攻击。
（3）安全的防雷设计
无线网桥内置平板天线不会引雷，避免雷击时不损坏连接的网络和主机设备。
（4）投资成本低、维护简单
不需架设线缆和挖沟布线，避免投入高昂的有线施工费用，施工周期短，维护简单方便，不影响建筑物内部、外部美观。
维护方便，可以通过任何一台电脑用Web浏览器对所有设备进行配置和管理。
（5）稳定可靠、高速率连接
无线设备一旦架设，运行稳定，最高连接速率可达54Mbps。

#4、项目实施阶段划分
为保障工程项目实施工作的顺利完成，我们将实施大体分成几个阶段，即：工程设计阶段、设备订货阶段、工程实施准备阶段、系统安装调试和测试阶段，按照系统设计技术建议书，详细分析用户需求，制定出具有技术细节的工程实施计划文档。

##4.1. 工程设计阶段
我们将和客户一起确认实施现场的具体情况进行工程设计细化工作，以便设计合适的无线网络系统，保证系统的先进性和合理性。

##4.2. 设备订货阶段
依据与最终用户签订的合同采购设备，组织供货以及工程所需要的各种工程耗材，为项目的按时开工打下基础。如合同采购的设备需要从国外进口，最短的供货周期为合同生效后8周。完成的标志为设备按时到达最终用户现场。

##4.3. 工程实施准备阶段
在合同生效,设备没有到达最终用户现场这段时间为工程实施小组的工程实施准备阶段。由技术人员出具项目技术实施细化文档给客户并进行商讨和改动，该技术细化文档应非常具体，如对IP地址的分配，VLAN的划分等等；客户方认可技术细化方案。

##4.4 系统安装调试阶段
在这个阶段为工程实施主要阶段，最终用户、XXX以及合作伙伴共同完成现场环境勘查、设备开箱验货、设备安装、设备加电测试、设备调试配置以及系统调试、系统初验测试等工作。具体工作如下：
1、现场检查:
最终用户与XX共同完成现场环境勘查：查看机房的位置、机房布置等情况，设计设备布置图，管槽分布图，楼内走线图。了解机房电源状况，环境指标，主要是检查机房环境能否满足设备安装要求,是否具备设备安装条件。
完成现场检查并就安装细节与客户取得共识后方可开始安装工作。
2、设备开箱验货：
当现场具备设备安装条件后，现场工程师应与最终用户代表一起依据合同对货物开箱验货和清点。完成这项工作后应与最终用户一起签署《开箱验货报告》。验货报告一式两份，作为双方交货的凭证以及支付设备款的凭证。其中一份交给最终用户，一份交给项目经理用于存档。
3、设备安装调试：
现场工程师对所有设备安装以及连接电缆进行检查、测试和调试并将结果以及已经完成的安装情况填写在《安装报告》及《实施记录》。安装报告一式两份，一份提交给最终用户，另一份提交给项目经理，以下为具体实施步骤：
（1）设备安装：
现场工程师应依据设备和环境平面布置图，完成设备上架固定等工作。原则上设备应固定在平面上并应满足抗震要求。设备安装完成后，现场工程师应并对所有设备安装以及连接电缆进行检查和测试。
（2）加电测试：
设备安装完成后，现场工程师应与最终用户的代表一起对设备进行加电测试，完成检查设备配置、设备予装软件等工作，具体加电测试工作如下：
a、 测试系统电源：测试检查系统中配置的电源能否满足设备对电源的要求（电源电压、可以提供的功率、频率、电源文波等）；
b、检查测试设备电源模块：测试检查设备本身电源模块输出电压是否正常；
c、检查设备能否正常启动。如果设备不能正常启动，网络工程师应查找故障原因。如果经努力仍无法排除故障，应及时通知项目经理；
d、如果加电后设备启动正常，现场工程师检查设备硬件配置、预装软件是否与设备说明书或系统设计手册相同。
(3)设备调试配置：
在加电测试完成后，现场工程师应逐一进行单个设备的调试配置，确保每一个设备都能正常的工作。设备调试包含的工作有：
a、根据设备厂商提供的设备技术手册测量调整设备硬件相关的技术参数，保证设备本身具有正常的工作状态；
b、根据设备厂商提供的设备技术手册进行设备的软件配置，并保证设备具有相应的功能；
c、应用软件的安装：在设备调试工作完成后，安装调试操作系统、网管软件等通用的软件平台。4、系统调试：
依据系统设计手册、网络拓扑图以及设备技术手册对整个网络或系统进行配置、统调。系统调试应包含：
对系统中的设备进行系统级的调试配置，保证系统或网络物理上是连通的；
依据系统设计手册为无线设备配置IP地址、SSID等相关技术参数，通过与有线网络部分的联调实现合法用户可以通过无线AP访问nternet的目的。

##4.5. 测试阶段
所有无线设备安装调试完毕以后，集中做系统的整体功能和性能测试工作，主要包括：无线设备的配置和监控、无线设备覆盖的范围和强度、检查是否存在覆盖盲点、通过MAC地址控制用户访问网络的权限等工作，完成后要填写相应的测试文档。
